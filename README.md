# 🦸 슈퍼 히어로 대모험

폰·태블릿·PC 어디서나 즐기는 **어린이용 반응형 웹 게임**이에요.
그림·소리·게임 로직이 전부 `index.html` 파일 하나 안에 들어 있어요.

## 🌐 지금 바로 플레이

**https://hero-adventure-eight.vercel.app**

## 🎮 게임 방법

- **⬅️ ➡️** : 좌우로 움직이기 (또는 화면의 ◀ ▶ 버튼)
- **스페이스바 / 점프 버튼** : 폴짝 점프 (바위를 뛰어넘어요)
- **Z / 발사 버튼 / 화면 터치** : 미사일 발사 (악당을 물리쳐요)
- 벽은 점프로 못 넘어요 → **구멍**을 찾아 좌우로 이동!
- 5개 레벨을 모두 통과하면 승리 🏆

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| 📱 반응형 | 폰·태블릿·PC 어떤 화면에서도 잘리지 않고 딱 맞아요 |
| 🏆 Top 5 점수판 | 게임이 끝나면 점수가 자동 저장되고 누적 순위를 보여줘요 |
| 🙈 이름 입력 없음 | 기기마다 별명(예: "반짝 펭귄 980")이 자동 생성돼요 |
| ⚙️ 난이도 설정 | 쉬움(하트 4) · 보통(하트 3) · 어려움(하트 2) 중 선택 |

## 🧱 무엇으로 만들어졌나요

- **화면/게임** : HTML + 자바스크립트 (`index.html` 한 파일)
- **점수 창고** : [Supabase](https://supabase.com) (점수 저장용 데이터베이스)
- **인터넷 배포** : [Vercel](https://vercel.com)

## 🛠️ 이어서 작업하려면 (개발을 몰라도 괜찮아요)

이 프로젝트는 **Claude에게 말로 부탁**하며 만들었어요. 다른 곳에서 이어서 하려면:

1. 이 저장소를 새 컴퓨터로 내려받아요.
2. Claude Code를 열고 이렇게 말하면 돼요. 예:
   - *"이 폴더의 index.html 게임을 열어서 보여줘"*
   - *"악당을 한 종류 더 추가해줘"*
   - *"난이도에 '아주 쉬움'을 하나 더 만들어줘"*
   - *"수정한 걸 다시 인터넷에 올려줘"*

명령어를 직접 칠 필요 없이, 하고 싶은 걸 우리말로 말하면 돼요. 🙂

## 🔑 참고 (점수판 창고 정보)

`index.html` 안에 Supabase **공개용 주소와 열쇠**가 들어 있어요.
이 값은 원래 브라우저에 공개되는 값이라 저장소에 있어도 안전해요.
