# 🦸 슈퍼 히어로 대모험

휴대폰·태블릿·PC 어디서나 즐길 수 있는 반응형 웹 게임입니다.
설치할 것 없이, 인터넷 주소만 열면 바로 플레이할 수 있습니다.

## 게임 방법
- **◀ ▶ (또는 화살표 키)**: 좌우로 움직이기
- **점프 버튼 (또는 스페이스바)**: 폴짝 뛰어서 바닥 장애물 넘기
- **발사 버튼 (또는 Z 키)**: 미사일을 쏴서 악당 물리치기
- 좁은 벽은 점프로 못 넘으니, 뚫린 구멍을 찾아 이동하세요!
- 5개의 레벨을 모두 통과하면 승리! 🏆

## 이 게임을 고치고 싶을 때
개발을 몰라도 괜찮습니다. Claude에게 이렇게 말씀하시면 됩니다. 예:
- "레벨을 하나 더 추가해줘"
- "악당을 더 느리게 만들어줘"
- "최고 점수를 저장하는 기능을 넣어줘"

## 파일 설명
- `index.html` : 인터넷 주소로 열면 자동으로 실행되는 대표 게임 파일
- `hero_adventure.html` : 원본 게임 파일 (내용은 위와 동일)
